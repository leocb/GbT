Dismemberment Mod

If you haven't had previous versions, use the addon format (paste
whole folder in addons)

If you have a previous version of Dismemberment Mod...
1. Open the Dismemberment Mod folder (you don't need to unzip)
2. Highlight ALL folders inside (Materials, LUA, and Data)
3. Right Click, Copy
4. Paste in Steam/SteamApps/"yourname"/garrysmod/garrysmod
5. If it asks you to merge, replace, or both, click yes (nothing will
be replaced even if it says)
